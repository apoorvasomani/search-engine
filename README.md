# README #

** Overview **

A search engine similar to Google Search built using the Google Custom Search API.

** Steps **

1) Run the following in terminal:
       
```
#!shell

> cd search_engine
> sh initialize_db.sh
> ./manage.py collectstatic
> ./manage.py runserver
```


2) Access it on [127.0.0.1:8000/google_search](Link URL)
